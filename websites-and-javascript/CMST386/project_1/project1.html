Project 1

How search engines work.

Search engines use specialized software called spiders to crawl millions of pages in order to read and index their content. Once a spider reads the content on the page, it begins to follow links to other pages. This process allows the search engine to find new pages all around the internet. Whenever a person conducts a query, the search engine compares keywords to those found in its database of millions of pages and displays the most relevant back to the person doing the query.

The Importance of Keywords and descriptions as meta tags

Keywords are the most important part of the algorithm in search engines. Keywords are what is stored inside the search engines database. Descriptions as meta tags help the search engine understand the content of the website. In google, for example, the description meta tag is displayed underneath the title of the search results.

The Four Elements that comprise writing relevant content links.

The four elements that comprise writing relevant content links are the following:
  1. The links should be descriptive and provide a simple idea of where the link is taking the user.
  2. The links should be a short phrase but descriptive.
  3. The links should be easy to distinguish from other text and background.
  4. Having internal links in the page is recommended, it allows google to navigate the page better.

Heading Hierarchy

Search engines look for heading hierarchy inside the website content and start to organize the website with the most important heading tag <h1> it then continues to the rest of the heading tags <h2>, <h3>, and so on. This heading hierarchy is why its important to organize the page content using the correct heading tag so that search engines detect the most important content on the page. For example
<body>
<h1>Main title of the page content</h1>
<h2>Secondary title of an article</h2>
<p>article content</p>
</body>
